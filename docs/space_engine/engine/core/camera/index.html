Crate camera

Expand description

This camera management system in Bevy allows defining various application states where the camera with the MainCamera component resides. Users can switch between different camera layers using commands, facilitating display management across different game scenes. The system supports various camera layer configurations, including clamping zoom, HDR, tonemapping, and render layers.
